Indian Railway Performance Dashboard
📌 Project Overview

The Indian Railway Performance Dashboard is a Power BI project designed to analyze railway operations and provide insights into revenue, passenger traffic, train occupancy, delays, and route performance. The dashboard transforms raw railway data into meaningful visualizations to support data-driven decision-making.

🎯 Objectives
Monitor railway performance using key KPIs.
Analyze passenger trends and occupancy rates.
Track train delays and punctuality.
Evaluate revenue generation across zones and routes.
Identify high-performing and underperforming railway segments.

📊 Key KPIs
Total Revenue
Total Passengers
Average Delay
Average Occupancy Rate
Total Trains

📈 Dashboard Features
Revenue Trend Analysis
Passenger Volume Analysis
Occupancy Rate by Zone
Revenue by Zone
Passenger Distribution by Ticket Class
Train Type Analysis
Route Performance Analysis
Occupancy vs Revenue Scatter Plot
Interactive Filters and Slicers
